## 第一性原理定位

K3 是**顶级前沿模型**（2.8T 参数、1M 上下文、对标 Claude Opus / GPT 顶级），不是 K2.7 的"贵价小升级"。

核心约束：MoA 卖点是"降本 90%"，K3 是仓库最贵模型（输出 $15/M，是 K2.7 的 3.75x、Flash 的 53x）。
- K3 只在 ~2% call volume 的**旗舰融合**环节 → 对混合单价冲击极小（$0.69 → $0.71）
- 这是它能进 MoA 而不毁"降本"卖点的**唯一物理基础**
- 用户必须**主动知道**价格和配额，否则视为坑用户

## 为什么是旗舰融合而非中级融合

- 中级·融合走 K2.7（便宜、高频），是降本链核心，不应被最贵模型顶替
- 旗舰融合只占 ~2%，K3 的 1M 上下文 + 顶级推理真正吃得下"三份架构方案→一份最优解"
- 不背叛"降本"卖点，反而把质量天花板推到第一梯队最前
